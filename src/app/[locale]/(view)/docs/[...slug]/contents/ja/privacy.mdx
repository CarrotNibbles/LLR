# プライバシーポリシー

1. **ポリシーの有効性と改訂**：本ポリシーはサービスの利用開始と同時に有効となり、改訂がある場合は告知されます。
2. **個人情報の収集と利用**：必要最低限の情報のみを収集し、事前に通知された範囲内で利用します。
3. **クッキーの使用**：クッキーを利用して訪問情報を収集し、ユーザーはブラウザで設定を調整することができます。
4. **情報の保管および廃棄**：目的達成後、即時に情報を削除しますが、法律に基づき一定のデータは保持されます。
5. **個人データの提供**：ユーザーの同意なく第三者にデータを提供することはありませんが、法律に基づき最小限の情報が提供される場合があります。
6. **データ処理の委託**：スムーズな運営のために一部の業務がDiscord Inc.、Supabase, Inc.、The Constant Company, LLC.、Cloudflare, Inc.に委託され、一部の情報が海外に移転されることがあります。
7. **苦情および救済措置**：苦情や救済措置については、[support@llr.app](mailto:support@llr.app)にご連絡ください。

**付録**：本ポリシーは2024年11月9日より有効です。
